# Como rodar o projeto

## 1 - clonar o projeto para a sua máquina
```
git clone https://github.com/rachidb5/test-frontEnd.git
```
## 2 - instalar dependencias
```
npm install
```
## 3 - executar
```
npm run serve
```

### porta
```
Você deve encontrar-lo no endereço: http://localhost:8080/
```
